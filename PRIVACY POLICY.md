# Privacy Policy

**Gemini Form Filler**
Last updated: April 2026

---

## Overview

Gemini Form Filler is a browser extension that uses the Google Gemini API to automatically fill Google Forms. This policy explains what data the extension accesses, how it is used, and what is never collected.

---

## Data the extension accesses

**Google Form content**
The extension reads the content of Google Forms pages you open — question text, answer options, and images attached to questions. This data is used solely to construct requests to the Gemini API and is never stored or transmitted anywhere else.

**Gemini API key**
Your API key is stored locally in your browser using `chrome.storage.local`. It is only transmitted to Google's Gemini API endpoint (`generativelanguage.googleapis.com`) to authenticate requests. It is never sent to any other server and never leaves your device for any other purpose.

**Selected model**
The name of the Gemini model you select is stored locally in `chrome.storage.local` and used only to configure API requests.

---

## Data the extension does NOT collect

- The extension does **not** collect, store, or transmit any personal information
- The extension does **not** use analytics, tracking, or telemetry of any kind
- The extension does **not** store form content, questions, or answers anywhere
- The extension does **not** have a backend server — all communication happens directly between your browser and Google's API
- The extension does **not** access any pages other than `https://docs.google.com/forms/*`

---

## Third-party services

The extension communicates with the following external services:

**Google Gemini API** (`generativelanguage.googleapis.com`)
Form question text and images are sent to the Gemini API to generate answers. This communication is subject to [Google's Privacy Policy](https://policies.google.com/privacy) and the [Gemini API Terms of Service](https://ai.google.dev/gemini-api/terms). Do not submit forms containing sensitive personal data using this extension.

**Google Forms image CDN** (`*.googleusercontent.com`)
When a question contains an image, the extension fetches that image to include it in the Gemini API request. The image is not stored and is only used for the duration of the API call.

---

## Local storage

The extension stores the following data locally on your device using `chrome.storage.local`:

| Data | Purpose |
|------|---------|
| Gemini API key | Authenticating requests to the Gemini API |
| Selected model name | Configuring API requests |
| List of available models | Displaying the model selector in the popup |

This data never leaves your device except as described above. You can remove it at any time by uninstalling the extension or clearing extension storage in your browser settings.

---

## AI-generated answers

Answers generated by the Gemini API are produced by an AI model and may be incorrect, incomplete, or inappropriate. The extension author takes no responsibility for the content of generated answers or the consequences of submitting them.
